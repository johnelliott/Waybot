Waybot Traffic Counter

The Project

Waybot is a home-brew traffic counter. The counter can be used with on-board memory or monitored live with a serial cable. A network-edge server and cloud server enable real-time monitoring in a web browser. The servers and apps are configured and deployed via Ansible scripts.

How it works

The counter: waterpoof case and coiled tube.Bicycles and cars roll over a pneumatic tube connected to an analog pressure sensor. The Arduino samples the voltage drop of the sensor and determines if a vehicle passed over the tube using an algorithm and sends data formatted as JSON for vehicle hits over the Arduino serial cable.

The Raspberry Pi running wb-upload. Wifi dongle with blue light connects to LAN. A Raspberry Pi acts as a bridge, taking the counter's JSON data from serial and making web requests to a Amazon EC2 server. This EC2 instance stores the data in a database and serves a web app that can sync with the hosted data.
